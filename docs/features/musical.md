# 音乐剧专区

[← 返回 README](../../README.md)

![音乐剧专区](../shots/musical.png)

`/category/musical`，顶栏直达。这里放的是舞台音乐剧（stage musical）：Hamilton、歌剧魅影、悲惨世界这一类，不是"带音乐的电影"。

## 数据从哪来

TMDB 没有舞台音乐剧分类，它的"音乐"类会混进演唱会电影和歌舞片。所以这个专区的数据源是一份人工精选清单：48 部百老汇与西区经典，覆盖当代（Hamilton、Dear Evan Hansen、Six、Hadestown）、韦伯系（歌剧魅影、猫、艾薇塔、日落大道）、桑德海姆（理发师陶德、拜访森林）、黄金年代（音乐之声、窈窕淑女、西区故事）、迪士尼舞台版（狮子王、冰雪奇缘）等。每部带中文译名、英文原名、首演年份和一句话简介。

海报按英文原名去 TMDB 借影视版或官摄版的封面，比如 Hamilton 用 2020 年 Disney+ 官摄那张。

## 今日推荐

每天从清单里轮换 10 部，出场顺序也每天洗牌。点卡片弹悬浮窗：先读简介，再从 B站官摄、腾讯视频、YouTube、BroadwayHD 里挑一个看。

## 我的收藏

管理员可以往收藏区加剧：

- **随机添加**：回答口味问题（经典正剧 / 动画歌舞 / 浪漫歌舞），从清单里挑 10 部还没收藏的
- **关键词添加**：搜索框里输入中英文名都行（"Hamilton"和"汉密尔顿"都命中），点选入库

收藏的卡片带青色"外站"角标，hover 可移除。普通用户能看能点，不能增删。

## 访问权限

音乐剧栏目走 scope 权限：未登录看不到入口，登录用户需要管理员在用户管理里勾选"音乐剧"才可见。boss 和 admin 不受限。
